# DB Driver Security Advisory Monitoring

Before DB driver introduction, the PR must include security advisory review evidence.

Required future checks:

- npm audit or equivalent advisory review.
- GitHub Advisory Database review.
- Known CVE review for the exact package/version.
- License review.
- Security update and patch response plan.
- Owner approval bound to the target commit.

This PR does not claim that any candidate package is advisory-clean or production-ready.

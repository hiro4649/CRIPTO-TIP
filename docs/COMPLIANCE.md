# Compliance

CRIPTO-TIP does not sell, swap, exchange, custody, cash out, or maintain internal crypto balances. It does not implement token sale, token swap, lottery, randomized reward, NFT prize, ranking reward, speculative reward, token price chart, or investment wording.

Tip is an AI character support effect, not investment. Affinity is non-transferable, non-refundable, non-cashable, and internal to IRIS.

On-chain event stores only minimal hashes and identifiers. Names, messages, YouTube author names, YouTube channel IDs, wallet labels, and comments are not stored on-chain.

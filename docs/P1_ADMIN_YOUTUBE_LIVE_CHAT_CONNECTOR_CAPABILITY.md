# P1 Admin YouTube Live Chat Connector Capability

This adds a read-only admin capability surface for the current fake YouTube Live Chat fixture client and fake connector loop boundaries.

The endpoint exposes only safe capability metadata: fixture-only mode, network disabled, OAuth not configured, real API execution false, fixture page support, cursor handoff support, and short-loop caps.

It does not expose raw config, OAuth tokens, API keys, client secrets, endpoints, headers, or private URLs. It does not call YouTube and does not claim connector, runtime, production, legal, or YouTube policy readiness.

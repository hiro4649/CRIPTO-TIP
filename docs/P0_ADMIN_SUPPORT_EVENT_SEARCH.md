# P0 Admin Support Event Search

This is local/internal admin support event search visibility only. It does not mutate support events and does not trigger side effects.

The search endpoint returns safe metadata for operators who need to find support events before opening a timeline, side-effect ledger, resend control, or adjustment control.

This does not create production Admin Console readiness, call real TTS, call real Live2D, call a real renderer, call real OBS, perform real WebSocket delivery, change web or overlay app behavior, use real DB, add Redis, Kafka, DB driver, package, or lockfile changes, expose raw messages, expose raw payloads, expose secrets, or claim runtime, production, legal, or YouTube policy readiness.

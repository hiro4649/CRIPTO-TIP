# Harness v1.2.0 High-Tier Repair Planning

`V120_HIGH_TIER_REPAIR_PLANNING` is a planning profile for high-risk repair work.

High-tier repair planning is not implementation approval. It does not create owner approval, GitHub approval review, merge authority, release authority, deploy authority, runtime readiness, production readiness, legal compliance, or YouTube policy compliance.

High-tier repairs should be split into small PRs with explicit scope, typed blockers, current-head evidence, and same-head required checks.

For CRIPTO-TIP, high-tier planning must preserve the product boundary:

- YouTube Super Chat payment is not replaced by crypto payment.
- IRIS Token Tip is not represented as YouTube Super Chat.
- IRIS does not custody user assets.
- IRIS does not provide token sale, exchange, cash-out, or internal balance.
- AI characters do not discuss token price or investment profit.
- Tip amount does not sell relationship status, ownership, or control.

# DB Driver License Review

License review is required before any DB driver dependency introduction.

This PR does not add a dependency and does not make a legal compliance claim. A future driver PR must record the license source, license text source, package version reviewed, and reviewer decision as safe evidence.

License review blockers:

- Unknown or ambiguous package license.
- License source not tied to the exact package version.
- License review mixed with unrelated package changes.
- Legal compliance claim without human owner/legal review.

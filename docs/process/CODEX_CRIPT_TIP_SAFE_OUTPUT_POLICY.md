# CRIPT-TIP Safe Output Policy

Feature name is CRIPT-TIP.
Repository name is CRIPTO-TIP.
Do not rename repository in code or docs without owner confirmation.

Safe output must not include raw logs, raw diffs, raw payloads, secrets, tokens, endpoints, private paths, wallet secrets, seed phrases, API keys, webhook secrets, RPC URLs, DB URLs, YouTube tokens, raw wallet addresses tied to viewer identity, or raw tx hashes tied to viewer identity.

No legal compliance claim, YouTube policy compliance claim, tax compliance claim, AML compliance claim, runtime readiness claim, production readiness claim, exchange service claim, custody claim, investment advice, price prediction, token sale, or fundraising claim is allowed.

Required policy statuses: criptTipExternalLinkSafetyStatus, criptTipViewerPrivacyBoundaryStatus, criptTipTxHashPrivacyStatus, criptTipRuntimeReadinessBoundaryStatus, criptTipProductionGoBoundaryStatus.
